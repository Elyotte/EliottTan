Lost Dog - Adventure game prototype

development time : 2 months alongside school project


Adventure game prototype where you play as an abandoned dog, a personal project developed alongside my studies in collaboration with a Game Designer teammate.


The goal of this project was to deepen my skills in Game Design and programming. I learned to implement rebindable input keys and create comprehensive Game Design documentation.

- Rebindable input system in Unity

- Game design documents.

- Level Design document.

- Fake screenshot creation.
